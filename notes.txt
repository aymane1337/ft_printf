va_list (object type) : this is a list that will contains all the dynamique arguments.

var_start (macro like-function) : it initialize everything, before moving throught our variable argument.
example : 
            int func(type format*, ...)
            var_list args;
            var_start(args, format);
va_arg (macro like-function) : allows us to access to the dynamique arguments from the variadic function , every time  we call va_arg we move to the next argument.
example :
            var_arg(args, char*) if "%s"
            var_arg(args, int) if "%d"
            and so on;
va_end : it cleans up the object that was initialized by calling va_start;

va_copy: it copy a specific arg;
